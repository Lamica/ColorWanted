# ColorWanted 赏色
这是一个**Windows**平台使用的屏幕取色器工具。有十六进制和RGB两种颜色值的显示。


>开发工具>=VS2010，运行环境需要>=.net4.0



## 快捷键
- **Alt+C** 复制十六进制颜色值
- **Alt+G** 复制RGB通道颜色值
- **Alt+H** 显示/隐藏颜色值显示窗口
- **Alt+F1** 开启/关闭跟随模式
